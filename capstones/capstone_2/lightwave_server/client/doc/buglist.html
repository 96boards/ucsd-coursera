LightWAVE bug list

  • Autoplay freezes after >10 minutes.
    It may have been possible to fill the trace pool with redundant copies of the same, or nearly the same, data segment, thereby locking out any attempt to fetch the next data segment. Changes in the prefetch strategy in version 0.46 make this very unlikely. Exit the browser and reopen the record if this problem recurs.
  • LightWAVE doesn't work in Internet Explorer.
    Yes, it does. But unless you have IE 9 or 10, you must install the free Google Chrome Frame plugin. The plugin is recommended but not required for IE 9.
