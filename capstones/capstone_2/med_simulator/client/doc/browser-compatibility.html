Browsers compatible with LightWAVE

[Last checked: 25 February 2013 (LightWAVE version 0.48)]

The LightWAVE client runs in a web browser and does not require any platform-specific components. The browser must support HTML5, JavaScript, and (for graphical output only) SVG. Recommended: Chrome, MSIE 9, and Safari currently provide the best performance because of their notably fast JavaScript engines. On older Windows systems, MSIE 7 or 8 (or even 6) with Google Chrome Frame are tolerable choices if you cannot install Chrome itself. On devices running Android 2.x and earlier, you will not have the Android 3 browser, and you may be unable to install Chrome 18; both Firefox 15 and Opera 12 are usable with limitations on Android 2.x, however, if you cannot update your Android platform.
